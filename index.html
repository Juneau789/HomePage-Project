About Me

Hello! My name is Harrison Juneau, and I’m a Computer Science student specializing in software engineering. I have experience in various programming languages, including Java, Python, and Swift, and I've worked on several projects ranging from mobile apps to web development. I’m passionate about coding, problem-solving, and learning new technologies. In addition to my technical skills, I have a strong interest in UX/UI design, which helps me create user-friendly applications.

When I’m not coding, you can find me fishing in the Louisiana marshes, exploring the outdoors, or working on personal projects. I’m always looking for ways to merge my love for technology with my passion for nature, and I hope to contribute to projects that have a meaningful impact on the environment and the community.

My Projects

Mobile

  • Catch Keeper

    A mobile app designed to help anglers log their fishing catches with GPS location, photos, and details like species and weight. Developed using Swift and SwiftUI.

  • Coding Game

    A fun and interactive mobile game that helps teach users about programming. Built with the React Native framework.

  • AutoCam App

    A dash camera app that uses the mobile device’s camera to capture videos at set intervals. Developed with Swift and SwiftUI for iOS devices.

Web

  • Personal Webpage

    A responsive personal webpage that showcases my portfolio, projects, and skills. Built using HTML, CSS, and JavaScript.

  • LSU Class Scheduler

    A webpage that allows students to find an optimal class schedule efficiently. Implemented using React.

Other

  • ZenScapeVR

    A virtual reality project integrating Arduino and a heart rate monitor to create an environment to help users build time management skills. Developed with Unity and C#.

  • Connect 4 AI

    An AI-powered Connect 4 game that uses the minimax algorithm with alpha-beta pruning to play against a human opponent. Implemented in Python with numpy and Pygame for the GUI.

  • Currency Converter

    A desktop application for converting currencies using fixed exchange rates. Built using Java and Java Swing for the GUI.
